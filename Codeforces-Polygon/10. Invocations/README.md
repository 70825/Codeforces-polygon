# Invocations

```General Info```, ```Statement```, ```Checker```, ```Validator```, ```Tests```, ```Solution files```을 완성한 뒤 채점을 해보는 곳입니다.

- ```General Info```에 설정한 시간 제한과 메모리 제한 기준으로 채점합니다.
- ```Checker```와 ```Validator```를 확인합니다.
- ```Soltuion files```에서 설정한 ```Type```과 일치하는지 확인합니다.

![image](https://user-images.githubusercontent.com/79046106/204021973-182902b2-2a97-4ab4-bd01-bf9cc541171e.png)

Want to run soltuions?를 클릭합니다.  

![image](https://user-images.githubusercontent.com/79046106/204022077-f5f253aa-7106-4b9f-8387-8e7a1e1cf01c.png)

- ```Solution to run```은 채점할 파일을 선택하는 곳입니다.
- ```Tests to run```은 채점할 데이터를 선택하는 곳입니다.

선택이 끝나면 ```Run jedgement``` 버튼을 클릭하면 됩니다.

![image](https://user-images.githubusercontent.com/79046106/204022421-eeb8856f-658d-43c6-9720-9af35eb62eb3.png)

이후 ```Views```를 클릭하고 새로고침하면 채점이 진행되는 과정을 확인할 수 있습니다.  

![image](https://user-images.githubusercontent.com/79046106/204022587-3c3b8293-a30b-43cd-9f8f-924994b355ab.png)

여기서 ```Solution files - Type```와 다른 결과가 나오는 경우엔 초록색 글자 대신 빨간색 글자가 나옵니다.  
사진에 ```WA```인데 빨간색 글자가 나오지 않은 경우는 ```Typo```를 ```Incorrect```로 설정해서 예상과 같기 때문입니다.
